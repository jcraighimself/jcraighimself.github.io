The Great
RGB
Colour Game

The game is quite simple. Choose either 3 or 6 squares with Easy or Hard, and try to guess which colour matches the RGB color given at the top.

The first number corresponds to how much Red is in the correct colour, between 0 (none at all) and 255 (as much as possible). The next number represents Green, and the last is Blue. Based on this, try to guess which colour square is the correctly matching one.
